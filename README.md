## KeenKeeper — Keep Your Friendships Alive

A responsive web app to help you track and maintain your 
friendships by reminding you to stay in touch...

## Technologies Used

- React.js
- React Router DOM
- Tailwind CSS + DaisyUI
- React Icons
- Recharts
- React Hot Toast

## Key Features

- Responsive Navbar with icons (Home, Timeline, Stats)
- Friend Cards Grid with status indicators
- Friend Details Page with Quick Check-In (Call, Text, Video)
- Timeline Page with filter (Call/Text/Video)
- Friendship Analytics Page with Pie Chart
- Loading animation, Toast notifications & 404 Page
- Fully responsive (Mobile, Tablet, Desktop)

- Beautiful UI matching Figma design
- 6–10 realistic friends data (JSON)
- Add interactions to Timeline
- Stats page with Recharts Pie Chart
- Timeline filter
- Toast notifications
- Responsive design
